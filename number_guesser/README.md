# Number Guessing Game

A CLI game where the user guesses a randomly generated number from 1 to 100.

## Features

- Random target number each round
- Input validation for non-integer and out-of-range values
- Hint feedback (`Too low` / `Too high`)
- Guess counter and replay option

## Concepts Practiced

- Loops and nested control flow
- Exception handling (`try`/`except`)
- Random number generation with `random.randint`
- Basic game state tracking

## Run

From the repository root:

```bash
python number-guesser/number_guesser.py
```
